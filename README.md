Layout y Organización:
[x] Se ha organizado el layout (sección "tarjetas") utilizando Flexbox.

[x] Redacta: ¿Qué ventajas encontraste al utilizar Flexbox o Grid en tu proyecto?
 La implementación de Flexbox me permitió una distribución dinámica y fluida de los elementos. La ventaja técnica principal fue el uso de flex-wrap, que facilita la adaptación del contenido a diferentes resoluciones sin necesidad de estructuras complejas, garantizando que los contenedores mantengan su alineación y proporción.
